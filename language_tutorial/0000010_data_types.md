# data types in python

## int / float

```python
int(float('5.6'))
int(5.6)
```

## str

```python
a_str = 'Hello'
a_str2 = "Hello"
a_str3: str = '''
    hello, World
'''
```

## bool

```python
a_bool: bool = True
a_bool_2: bool = False
```

## list

```python
a_list = []
a_list_2: list[int] = [1, 2, 3]
```

## dict

```python
a_dict: dict[str, int] = {}
```
